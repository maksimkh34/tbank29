Transactions

Синхронизация транзакций

Получение списка транзакций, измененных после указанного момента времени. Если параметр since не передан, возвращается полный список доступных транзакций. Сравнение выполняется по полю updatedAt по правилу updatedAt > since.

Authorizations:
BearerAuth
query Parameters
since
string <date-time>
Example: since=2026-04-16T17:00:00Z

ISO 8601 дата-время последней синхронизации по полю updatedAt

Responses

API Server

https://api.finassist.com/api/v1/transactions
