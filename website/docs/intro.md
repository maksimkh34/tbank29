# О проекте FinAssist

**FinAssist** — это современный сервис для учета личных финансов, предназначенный для пользователей, которые ценят свое время и стремятся к финансовой грамотности. Система объединяет мобильное приложение и мощную серверную часть для автоматизации рутинных задач по учету расходов и доходов.

## Ключевые возможности

- **Единое окно для всех финансов**: Поддержка банковских карт, криптокошельков, наличных и других активов.
- **Интеграция с банками**: Полуавтоматический импорт транзакций через API.
- **ИИ-ассистент**: Виртуальный финансист, который анализирует ваши траты и дает персональные советы.
- **Умная категоризация**: Алгоритмы машинного обучения, которые со временем учатся автоматически определять категории ваших расходов.
- **Кросс-платформенная синхронизация**: Ваши данные всегда под рукой на любом устройстве.

## Целевая аудитория

1. **Индивидуальные пользователи**: Желающие навести порядок в бюджете.
2. **Семьи**: Для ведения общего бюджета и прозрачного планирования.
3. **Фрилансеры и самозанятые**: Для учета непостоянных доходов и разделения личных и рабочих финансов.
4. **Студенты**: Которым нужен простой инструмент для освоения навыков финансовой грамотности.

## Бизнес-цели (MVP)

- Достичь **5000 активных пользователей** через 6 месяцев после запуска.
- Обеспечить автоматическое определение категорий для **80% транзакций**.
- Выйти на самоокупаемость в течение полугода после релиза.
